一、表达式和语句

1、表达式

一个表达式可以产生一个值，有可能是运算(操作数和运算符组成)、函数调用、有可能是字面量。表达式可以放在任何需要值的地方。

5 –>字面量 true “Hello World”

2、语句

语句可以理解为一个行为，输出语句和赋值就是典型的语句。一个程序有很多个语句组成，一般情况下;分割一个一个的语句

二、相关概念

1、语句结构分类

​ 语句结构：语句执行的流程。

​ 程序的流程控制有三种:顺序结构,选择结构,循环结构。

2、语句块

​ 已经学的语句块：主程序框架【类主体】+main主入口框架【main主体】

​ 语句块，就是“结构名称”包含的一段{ }： { 代表开始，}代表结束，同一个语句块要注意对齐，嵌套的要缩进。

3、逻辑运算符

​ 可以合并多个条件表达式

​ &&：且 (htmlScore>90) && (javaSocre>90)，都要满足

​ ||：或 (htmlScore>90) || (javaSocre>90)，满足一个就可以

​ ！：非/取反 !(htmlScore>90) 分数小于等于90

三、选择结构一

if选择语句

根据条件判断之后再做处理的一种语法结构。

1）基本if选择语句

注：表达式必须结果布尔值true/false，代表条件的真假。

  • 执行流程
    • 首先判断关系表达式看其结果是true还是false
    • 如果是true就执行语句体
    • 如果是false就不执行语句体

2）if-else选择语句

  • 执行流程
    • 首先判断关系表达式看其结果是true还是false
    • 如果是true就执行语句体1
    • 如果是false就执行语句体2

3）多重if选择语句

  • 执行流程
    • 首先判断关系表达式1看其结果是true还是false
    • 如果是true就执行语句体1
    • 如果是false就继续判断关系表达式2看其结果是true还是false
    • 如果是true就执行语句体2
    • 如果是false就继续判断关系表达式…看其结果是true还是false
    • …
    • 如果没有任何关系表达式为true，就执行语句体n+1。

4）嵌套if选择语句

四、选择结构二

switch选择结构

1）语法结构

  • 执行流程
    • 首先计算出表达式的值
    • 其次，和case依次比较，一旦有对应的值，就会执行相应的语句，在执行的过程中，遇到break就会结 束。
    • 最后，如果所有的case都和表达式的值不匹配，就会执行default语句体部分，然后程序结束掉。

2）产生随机数

 int random = (int) (Math.random() * 10);   结果是0-10之间的整数

3）处理系统异常

​ xx.hasNextInt() 如果xx扫描器输入的是数字，结果为true

4）switch和多重if区别

相同点：都是用来处理多分支条件的结构

不同点：

​ switch选择结构，【只】能处理【等值条件判断】的情况。

​ 多重if选择结构： 【不仅】能处理【等值条件判断】，也能处理【连续区间条件判断】
总结：等值判断用switch，区间判断用if。
